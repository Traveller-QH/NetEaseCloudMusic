# 网易云音乐登录功能使用说明

## 功能概述
本项目实现了完整的登录功能，支持三种登录方式：
1. **手机号密码登录** - 使用手机号和密码登录
2. **手机验证码登录** - 使用手机号和短信验证码登录
3. **二维码登录** - 使用网易云音乐APP扫码登录

## 功能特点
- ✅ 支持国际手机号登录（国家区号选择）
- ✅ 验证码倒计时功能
- ✅ 二维码自动生成和状态轮询
- ✅ 登录状态持久化存储
- ✅ 页面刷新后自动恢复登录状态
- ✅ 完整的用户信息展示

## 使用方法

### 1. 访问登录页面
在"我的"页面点击未登录状态下的用户区域，会跳转到登录页面。

### 2. 手机号密码登录
1. 选择"密码登录"标签
2. 选择国家/地区（默认中国 +86）
3. 输入手机号
4. 输入密码
5. 点击"登录"按钮

### 3. 手机验证码登录
1. 选择"验证码登录"标签
2. 选择国家/地区
3. 输入手机号
4. 点击"获取验证码"按钮
5. 输入收到的验证码
6. 点击"登录"按钮

### 4. 二维码登录
1. 选择"扫码登录"标签
2. 自动显示二维码
3. 使用网易云音乐APP扫描二维码
4. 在手机上确认登录
5. 自动完成登录

## 技术实现

### 数据持久化
登录成功后，以下信息会自动持久化存储：
- 用户基本信息（昵称、头像、关注数等）
- 账户信息（包含重要的 `baoyueVersion` 字段）
- 登录状态和时间戳
- Token和Cookie信息

### 状态管理
使用 Pinia 进行状态管理：
- `isLoggedIn` - 登录状态
- `userInfo` - 用户基本信息
- `accountInfo` - 账户详细信息
- `cookie` - 登录凭证

### 自动恢复
页面刷新时会自动：
1. 从本地存储恢复登录状态
2. 验证登录凭证有效性
3. 更新用户信息
4. 保持登录状态

## 注意事项
1. 验证码有60秒发送间隔限制
2. 二维码有效期为5分钟
3. 登录状态默认保存7天
4. 支持页面刷新后自动恢复登录
